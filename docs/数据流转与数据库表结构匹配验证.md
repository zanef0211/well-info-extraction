```python
class Well(Base):
```

## 二、数据流转流程详解

```
用户上传文件
    ↓
保存文件到存储
    ↓
AI处理 (OCR → 分类 → 抽取 → 校验)
    ↓
【步骤1】创建/更新 Well 记录 (确保外键约束有效)
    ↓
【步骤2】创建 Document 记录 (status='processing')
    ↓
【步骤3】批量插入 ExtractedData 记录
    ↓
【步骤4】插入 QualityReport 记录
    ↓
【步骤5】批量插入 ValidationResult 记录
    ↓
【步骤6】插入 ProcessingLog 记录
    ↓
【步骤7】更新 Document 记录 (status='success')
    ↓
返回API响应 ✅ 数据已持久化
```

## 三、CRUD 实现与数据库表结构匹配验证

### 步骤1: 创建/更新 Well 记录

#### 数据库表结构要求
- ✅ 主键: `id` (自增整数)
- ✅ 业务键: `well_no` (唯一字符串)
- ✅ 外键引用: `documents.well_no`, `extracted_data.well_no`, `quality_reports.well_no`, `processing_logs.well_no`

#### CRUD 实现 (db/crud_fixed.py)
```python
class WellCRUD:
    @staticmethod
    def get_or_create_well(db: Session, well_no: str) -> Well:
        # 先尝试获取已存在的记录
        well = db.query(Well).filter(Well.well_no == well_no).first()
        
        # 如果不存在，创建新记录
        if not well:
            well = Well(
                well_no=well_no,  # ✅ 使用 well_no (字符串)
                status='active'
            )
            db.add(well)
            db.commit()
            db.refresh(well)  # ✅ 刷新后获得自增的 id
        
        return well  # ✅ 返回 Well 对象，包含自增的 id
```

#### 验证结果
- ✅ 使用 `well_no` 字符串查询/创建
- ✅ 提交后自动获得自增的 `id` (整数)
- ✅ 返回完整的 Well 对象
- ✅ **匹配数据库表结构**

```python
class ExtractedDataCRUD:
    @staticmethod
    def batch_create_extracted_data(
        db: Session,
        extracted_data_list: List[Dict[str, Any]]
    ) -> List[ExtractedData]:
        records = []
        for data in extracted_data_list:
            # ✅ 确保 document_id 是整数
            doc_id = data['document_id']
            if not isinstance(doc_id, int):
                raise TypeError(f"document_id 必须是整数，但得到 {type(doc_id)}")
            
            record = ExtractedData(
                document_id=doc_id,  # ✅ 外键: 整数 (关联 documents.id)
                well_no=data['well_no'],  # ✅ 外键: 字符串
                field_name=data['field_name'],
                field_value=data.get('field_value'),
                field_type=data.get('field_type'),
                confidence=data.get('confidence'),
                is_valid=data.get('is_valid', True),
                validation_errors=data.get('validation_errors'),  # ✅ JSONB类型
                source=data.get('source', 'llm')
            )
            records.append(record)
        
        db.add_all(records)
        db.commit()
        
        # 刷新所有记录以获取ID
        for record in records:
            db.refresh(record)
        
        return records
```
