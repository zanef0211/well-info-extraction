## 💾 数据库表设计

### 1. Well 表 (核心表)

```python
class Well(Base):
    __tablename__ = "wells"
    
    id = Column(Integer, primary_key=True)
    well_no = Column(String(50), unique=True, nullable=False, index=True)
    well_name = Column(String(200))
    oilfield = Column(String(100))
    block = Column(String(100))
    well_type = Column(String(50))      # 探井/开发井/注水井等
    well_pattern = Column(String(50))   # 直井/定向井/水平井
    latitude = Column(Numeric(10, 6))
    longitude = Column(Numeric(10, 6))
    drill_date = Column(Date)
    completion_date = Column(Date)
    status = Column(String(20), default='active')
    
    # 关系
    documents = relationship("Document", cascade="all, delete-orphan")
    extracted_data = relationship("ExtractedData", cascade="all, delete-orphan")
    quality_reports = relationship("QualityReport", cascade="all, delete-orphan")
    processing_logs = relationship("ProcessingLog", cascade="all, delete-orphan")
```

**关键点**:
- ✅ `well_no` 是唯一标识
- ✅ 所有业务表都通过外键或 `well_no` 字段关联
- ✅ 级联删除确保数据一致性

---

### 2. Document 表

```python
class Document(Base):
    __tablename__ = "documents"
    
    id = Column(Integer, primary_key=True)
    well_no = Column(String(50), nullable=False, index=True)
    filename = Column(String(255), nullable=False)
    document_type = Column(String(50))
    category = Column(String(50))
    status = Column(String(20), default='pending')
    
    # 关系
    well = relationship("Well", back_populates="documents")
    extracted_data = relationship("ExtractedData", cascade="all, delete-orphan")
    quality_reports = relationship("QualityReport", cascade="all, delete-orphan")
```

**关键点**:
- ✅ `well_no` 是必需字段
- ✅ 建立 `well_no` 索引便于查询
- ✅ 所有提取数据都通过文档关联到井号

---

### 3. ExtractedData 表

```python
class ExtractedData(Base):
    __tablename__ = "extracted_data"
    
    id = Column(Integer, primary_key=True)
    document_id = Column(Integer, ForeignKey('documents.id'), nullable=False)
    well_no = Column(String(50), nullable=False, index=True)
    field_name = Column(String(100), nullable=False)
    field_value = Column(Text)
    confidence = Column(Numeric(5, 2))
    
    # 关系
    document = relationship("Document")
    well = relationship("Well")
```

**关键点**:
- ✅ 通过 `document_id` 关联到文档
- ✅ 冗余 `well_no` 字段便于直接查询
- ✅ 建立双重索引 (`document_id`, `well_no`)

## 📝 使用示例

### 示例1: 查询某口井的完整信息

```python
from db.models import Well
from db.database import SessionLocal

db = SessionLocal()

# 查询井
well = db.query(Well).filter(Well.well_no == "XX-1-1").first()

# 访问该井的所有数据
print(f"井号: {well.well_no}")
print(f"文档数: {len(well.documents)}")
print(f"提取数据条数: {len(well.extracted_data)}")
print(f"质量评估: {len(well.quality_reports)}")
print(f"处理日志: {len(well.processing_logs)}")

db.close()
```

---

### 示例2: 批量导入多井资料

```python
from db.models import Well, Document
from db.database import SessionLocal

db = SessionLocal()

# 多井资料
files = [
    {"filename": "XX-1-1-日报.pdf", "well_no": "XX-1-1"},
    {"filename": "XX-1-2-日报.pdf", "well_no": "XX-1-2"},
]

for file_info in files:
    # 确保井存在
    well = db.query(Well).filter(Well.well_no == file_info["well_no"]).first()
    if not well:
        well = Well(well_no=file_info["well_no"])
        db.add(well)

    # 创建文档
    doc = Document(
        well_no=file_info["well_no"],
        filename=file_info["filename"]
    )
    db.add(doc)

db.commit()
db.close()
```

### 示例3: 按井号统计

```python
from db.models import Well, Document
from db.database import SessionLocal
from sqlalchemy import func

db = SessionLocal()

# 统计每口井的文档数量
result = db.query(
    Well.well_no,
    func.count(Document.id).label('doc_count')
).join(Document).group_by(Well.well_no).all()

for well_no, count in result:
    print(f"{well_no}: {count} 份文档")

db.close()
```
