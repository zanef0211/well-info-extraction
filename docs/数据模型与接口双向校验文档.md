## 一、数据流转方案设计

### 1.1 方案选择

**方案对比**:

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **方案A: 先落库再返回** | 数据持久化,可追溯,支持后续查询 | 响应较慢,数据库事务复杂 | 生产环境,需要审计 |
| **方案B: 不落库直接返回** | 响应快速,无数据库开销 | 数据不持久,无法追溯 | 临时处理,快速验证 |
| **方案C: 混合模式** | 兼顾性能和持久化,灵活配置 | 实现稍复杂 | 推荐方案 |

**推荐方案: 方案C - 混合模式**

```
┌─────────────────────────────────────────────────────────────┐
│                      数据流转架构                              │
└─────────────────────────────────────────────────────────────┘

1. 文档上传
   ↓
2. 保存文件到存储 (storage/uploads/)
   ↓
3. 处理文档 (OCR → 分类 → 抽取 → 校验)
   ↓
4. 创建Document记录 (status='processing')
   ↓
5. 插入ExtractedData记录 (批量插入)
   ↓
6. 插入QualityReport记录
   ↓
7. 插入ValidationResult记录
   ↓
8. 更新Document记录 (status='success')
   ↓
9. 返回API响应 (包含处理结果)
   ↓
10. (可选) 异步更新统计信息

关键点:
- 文档处理失败时不写入ExtractedData,但Document记录保存(便于追溯)
- 使用数据库事务保证一致性
- 批量插入提高性能
- 提供配置选项: save_to_db (默认true)
```

```python
from typing import List, Dict
from config.field_schemas import FIELD_SCHEMAS
from db.models import ExtractedData


def validate_field_definition(field_name: str, category: str) -> bool:
    """
    验证字段定义是否在字段库中

    Args:
        field_name: 字段名
        category: 分类

    Returns:
        是否有效
    """
    # 获取该分类的所有字段
    all_fields = FIELD_SCHEMAS.get_all_fields(category)
    field_names = [f.name for f in all_fields]

    return field_name in field_names


def validate_field_value(
    field_name: str,
    field_value: any,
    category: str
) -> tuple[bool, List[str]]:
    """
    根据字段库定义验证字段值

    Args:
        field_name: 字段名
        field_value: 字段值
        category: 分类

    Returns:
        (是否有效, 错误信息列表)
    """
    errors = []

    # 1. 获取字段定义
    all_fields = FIELD_SCHEMAS.get_all_fields(category)
    field_def = next((f for f in all_fields if f.name == field_name), None)

    if not field_def:
        errors.append(f"字段'{field_name}'不在分类'{category}'的字段库中")
        return False, errors

    # 2. 检查必填字段
    if field_def.required and (field_value is None or field_value == ""):
        errors.append(f"必填字段'{field_name}'为空")

    # 3. 检查字段类型
    if field_value is not None and field_value != "":
        if field_def.data_type == "float":
            try:
                num_value = float(field_value)
            except (ValueError, TypeError):
                errors.append(f"字段'{field_name}'应为浮点数类型")

        elif field_def.data_type == "int":
            try:
                int_value = int(field_value)
            except (ValueError, TypeError):
                errors.append(f"字段'{field_name}'应为整数类型")

        elif field_def.data_type == "date":
            # 日期格式检查在FieldValidator中处理
            pass

    # 4. 检查数值范围
    if field_value is not None and field_value != "":
        try:
            num_value = float(field_value)
            if field_def.min_value is not None and num_value < field_def.min_value:
                errors.append(
                    f"字段'{field_name}'值过小: {num_value} < {field_def.min_value}"
                )
            if field_def.max_value is not None and num_value > field_def.max_value:
                errors.append(
                    f"字段'{field_name}'值过大: {num_value} > {field_def.max_value}"
                )
        except (ValueError, TypeError):
            pass

    return len(errors) == 0, errors
```

```python
from sqlalchemy import inspect
from db.models import Base, Well, Document, ExtractedData, QualityReport


def validate_database_schema() -> Dict[str, List[str]]:
    """
    验证数据库表结构是否符合设计规范

    Returns:
        验证结果: {表名: [警告/错误列表]}
    """
    results = {}

    # 1. 检查表是否存在
    inspector = inspect(engine)
    existing_tables = inspector.get_table_names()

    required_tables = [
        'wells', 'documents', 'extracted_data',
        'quality_reports', 'validation_results',
        'review_records', 'processing_logs', 'users', 'statistics'
    ]

    for table in required_tables:
        if table not in existing_tables:
            results[table] = [f"表'{table}'不存在"]

    # 2. 检查wells表
    if 'wells' in existing_tables:
        results['wells'] = _validate_wells_table(inspector)

    # 3. 检查documents表
    if 'documents' in existing_tables:
        results['documents'] = _validate_documents_table(inspector)

    # 4. 检查extracted_data表
    if 'extracted_data' in existing_tables:
        results['extracted_data'] = _validate_extracted_data_table(inspector)

    return results


def _validate_wells_table(inspector) -> List[str]:
    """验证wells表结构"""
    issues = []

    # 检查必填列
    required_columns = ['id', 'well_no', 'created_at', 'updated_at']
    columns = [col['name'] for col in inspector.get_columns('wells')]

    for col in required_columns:
        if col not in columns:
            issues.append(f"缺少必填列: {col}")

    # 检查唯一约束
    constraints = inspector.get_unique_constraints('wells')
    unique_columns = []
    for constraint in constraints:
        unique_columns.extend(constraint.get('column_names', []))

    if 'well_no' not in unique_columns:
        issues.append("well_no列应该有唯一约束")

    # 检查索引
    indexes = inspector.get_indexes('wells')
    indexed_columns = []
    for idx in indexes:
        indexed_columns.extend(idx.get('column_names', []))

    required_indexes = ['well_no', 'oilfield', 'block', 'status']
    for idx_col in required_indexes:
        if idx_col not in indexed_columns:
            issues.append(f"缺少索引: {idx_col}")

    return issues


def _validate_documents_table(inspector) -> List[str]:
    """验证documents表结构"""
    issues = []

    # 检查必填列
    required_columns = ['id', 'well_no', 'filename', 'document_id']
    columns = [col['name'] for col in inspector.get_columns('documents')]

    for col in required_columns:
        if col not in columns:
            issues.append(f"缺少必填列: {col}")

    # 检查外键
    foreign_keys = inspector.get_foreign_keys('documents')
    fk_to_wells = any(
        fk['referred_table'] == 'wells'
        for fk in foreign_keys
    )

    if not fk_to_wells:
        issues.append("documents表应该有指向wells表的外键")

    return issues


def _validate_extracted_data_table(inspector) -> List[str]:
    """验证extracted_data表结构"""
    issues = []

    # 检查必填列
    required_columns = [
        'id', 'document_id', 'well_no', 'field_name'
    ]
    columns = [col['name'] for col in inspector.get_columns('extracted_data')]

    for col in required_columns:
        if col not in columns:
            issues.append(f"缺少必填列: {col}")

    # 检查唯一约束
    constraints = inspector.get_unique_constraints('extracted_data')
    has_document_field_unique = any(
        'document_id' in constraint.get('column_names', []) and
        'field_name' in constraint.get('column_names', [])
        for constraint in constraints
    )

    if not has_document_field_unique:
        issues.append("应该有(document_id, field_name)的唯一约束")

    # 检查外键
    foreign_keys = inspector.get_foreign_keys('extracted_data')
    fk_to_documents = any(
        fk['referred_table'] == 'documents'
        for fk in foreign_keys
    )

    if not fk_to_documents:
        issues.append("extracted_data表应该有指向documents表的外键")

    return issues
```

```python
def validate_extracted_data_integrity(
    well_no: str,
    category: str
) -> Dict[str, any]:
    """
    验证提取数据的完整性

    Args:
        well_no: 井号
        category: 分类

    Returns:
        验证结果
    """
    # 获取该井该分类的所有文档
    documents = db.query(Document).filter(
        Document.well_no == well_no,
        Document.category == category
    ).all()

    if not documents:
        return {
            "valid": True,
            "message": "无数据",
            "issues": []
        }

    # 获取所有提取的字段
    all_extracted_fields = []
    for doc in documents:
        for data in doc.extracted_data:
            all_extracted_fields.append(data.field_name)

    # 获取字段库中该分类的所有字段
    required_fields = FIELD_SCHEMAS.get_required_fields(category)
    all_fields = FIELD_SCHEMAS.get_field_names(category)

    # 检查必填字段
    missing_required = [
        field for field in required_fields
        if field not in all_extracted_fields
    ]

    # 检查是否有无效字段
    invalid_fields = [
        field for field in set(all_extracted_fields)
        if field not in all_fields
    ]

    issues = []
    if missing_required:
        issues.append({
            "type": "missing_required",
            "fields": missing_required
        })

    if invalid_fields:
        issues.append({
            "type": "invalid_fields",
            "fields": invalid_fields
        })

    return {
        "valid": len(issues) == 0,
        "message": "验证通过" if len(issues) == 0 else "发现问题",
        "issues": issues,
        "missing_required": missing_required,
        "invalid_fields": invalid_fields
    }
```

```python
from sqlalchemy.orm import Session
from typing import List, Optional, Dict, Any
from datetime import datetime

from db.models import Well, Document, ExtractedData, QualityReport, ValidationResult
from db.database import get_db


class WellCRUD:
    """井信息CRUD操作"""

    @staticmethod
    def get_or_create_well(db: Session, well_no: str) -> Well:
        """
        获取或创建井记录

        Args:
            db: 数据库会话
            well_no: 井号

        Returns:
            Well对象
        """
        well = db.query(Well).filter(Well.well_no == well_no).first()
        if not well:
            well = Well(well_no=well_no, status='active')
            db.add(well)
            db.commit()
            db.refresh(well)
        return well

    @staticmethod
    def get_well(db: Session, well_no: str) -> Optional[Well]:
        """获取井信息"""
        return db.query(Well).filter(Well.well_no == well_no).first()

    @staticmethod
    def update_well_info(db: Session, well_no: str, data: Dict[str, Any]) -> Optional[Well]:
        """更新井信息"""
        well = WellCRUD.get_well(db, well_no)
        if well:
            for key, value in data.items():
                if hasattr(well, key):
                    setattr(well, key, value)
            well.updated_at = datetime.utcnow()
            db.commit()
            db.refresh(well)
        return well


class DocumentCRUD:
    """文档CRUD操作"""

    @staticmethod
    def create_document(
        db: Session,
        well_no: str,
        document_id: str,
        filename: str,
        file_path: str,
        file_size: int,
        category: Optional[str] = None,
        doc_category: Optional[str] = None,
    ) -> Document:
        """
        创建文档记录

        Args:
            db: 数据库会话
            well_no: 井号
            document_id: 文档ID
            filename: 文件名
            file_path: 文件路径
            file_size: 文件大小
            category: 分类
            doc_category: 二级分类

        Returns:
            Document对象
        """
        # 获取或创建井记录
        well = WellCRUD.get_or_create_well(db, well_no)

        document = Document(
            well_no=well_no,
            document_id=document_id,
            filename=filename,
            file_path=file_path,
            file_size=file_size,
            file_extension=filename.split('.')[-1].lower() if '.' in filename else None,
            category=category,
            doc_category=doc_category,
            status='processing',
            upload_date=datetime.utcnow().date()
        )
        db.add(document)
        db.commit()
        db.refresh(document)
        return document

    @staticmethod
    def update_document_status(
        db: Session,
        document_id: str,
        status: str
    ) -> Optional[Document]:
        """更新文档状态"""
        document = db.query(Document).filter(
            Document.document_id == document_id
        ).first()
        if document:
            document.status = status
            document.updated_at = datetime.utcnow()
            db.commit()
            db.refresh(document)
        return document

    @staticmethod
    def get_document(db: Session, document_id: str) -> Optional[Document]:
        """获取文档"""
        return db.query(Document).filter(
            Document.document_id == document_id
        ).first()

    @staticmethod
    def get_documents_by_well(db: Session, well_no: str) -> List[Document]:
        """获取井的所有文档"""
        return db.query(Document).filter(
            Document.well_no == well_no
        ).all()


class ExtractedDataCRUD:
    """提取数据CRUD操作"""

    @staticmethod
    def batch_insert_extracted_data(
        db: Session,
        document_id: int,
        well_no: str,
        extracted_fields: Dict[str, Any],
    ) -> List[ExtractedData]:
        """
        批量插入提取数据

        Args:
            db: 数据库会话
            document_id: 文档ID
            well_no: 井号
            extracted_fields: 提取的字段字典

        Returns:
            ExtractedData对象列表
        """
        extracted_data_list = []

        for field_name, field_data in extracted_fields.items():
            # 提取字段值和置信度
            if isinstance(field_data, dict):
                value = field_data.get('value')
                confidence = field_data.get('confidence', 0.0)
                source = field_data.get('source')
                validated = field_data.get('validated', True)
            else:
                value = field_data
                confidence = 0.0
                source = None
                validated = True

            # 判断字段类型
            field_type = 'string'
            if isinstance(value, (int, float)):
                field_type = 'float' if isinstance(value, float) else 'int'

            extracted_data = ExtractedData(
                document_id=document_id,
                well_no=well_no,
                field_name=field_name,
                field_value=str(value) if value is not None else None,
                field_type=field_type,
                confidence=float(confidence) if confidence else 0.0,
                is_valid=validated,
                source=source or 'llm',
            )
            extracted_data_list.append(extracted_data)

        # 批量插入
        db.bulk_save_objects(extracted_data_list)
        db.commit()

        return extracted_data_list

    @staticmethod
    def get_extracted_data_by_well(
        db: Session,
        well_no: str,
        field_name: Optional[str] = None
    ) -> List[ExtractedData]:
        """获取井的提取数据"""
        query = db.query(ExtractedData).filter(
            ExtractedData.well_no == well_no
        )

        if field_name:
            query = query.filter(ExtractedData.field_name == field_name)

        return query.all()


class QualityReportCRUD:
    """质量报告CRUD操作"""

    @staticmethod
    def create_quality_report(
        db: Session,
        document_id: int,
        well_no: str,
        quality_metrics: Dict[str, float],
        issues: List[Dict[str, str]],
        suggestions: List[str]
    ) -> QualityReport:
        """
        创建质量报告

        Args:
            db: 数据库会话
            document_id: 文档ID
            well_no: 井号
            quality_metrics: 质量指标
            issues: 问题列表
            suggestions: 建议列表

        Returns:
            QualityReport对象
        """
        # 计算质量等级
        overall_score = quality_metrics.get('overall_score', 0.0)
        if overall_score >= 0.9:
            quality_level = 'excellent'
        elif overall_score >= 0.8:
            quality_level = 'good'
        elif overall_score >= 0.6:
            quality_level = 'acceptable'
        else:
            quality_level = 'poor'

        quality_report = QualityReport(
            document_id=document_id,
            well_no=well_no,
            completeness=float(quality_metrics.get('completeness', 0.0)),
            accuracy=float(quality_metrics.get('accuracy', 0.0)),
            consistency=float(quality_metrics.get('consistency', 0.0)),
            confidence=float(quality_metrics.get('confidence', 0.0)),
            overall_score=float(overall_score),
            quality_level=quality_level,
            issues=issues,
            suggestions=suggestions,
            validated_at=datetime.utcnow()
        )

        db.add(quality_report)
        db.commit()
        db.refresh(quality_report)

        return quality_report


class ValidationResultCRUD:
    """验证结果CRUD操作"""

    @staticmethod
    def batch_insert_validation_results(
        db: Session,
        document_id: int,
        validation_results: Dict[str, Any]
    ) -> List[ValidationResult]:
        """
        批量插入验证结果

        Args:
            db: 数据库会话
            document_id: 文档ID
            validation_results: 验证结果字典

        Returns:
            ValidationResult对象列表
        """
        validation_list = []

        for field_name, result in validation_results.items():
            if isinstance(result, dict):
                passed = result.get('is_valid', True)
                errors = result.get('errors', [])
                warnings = result.get('warnings', [])

                # 转换错误信息
                error_messages = []
                for error in errors:
                    error_messages.append({
                        'level': 'error',
                        'message': error
                    })
                for warning in warnings:
                    error_messages.append({
                        'level': 'warning',
                        'message': warning
                    })

                validation_result = ValidationResult(
                    document_id=document_id,
                    field_name=field_name,
                    passed=passed,
                    error_message=str(error_messages) if error_messages else None,
                    suggestion=None  # 可后续添加
                )
                validation_list.append(validation_result)

        # 批量插入
        db.bulk_save_objects(validation_list)
        db.commit()

        return validation_list
```

### 3.2 完整的数据持久化流程

```python
from db.crud import (
    WellCRUD, DocumentCRUD, ExtractedDataCRUD,

from db.database import get_db
from config.settings import settings


def save_processing_result_to_database(

    """

    1. 创建/获取Well记录
    2. 创建Document记录

    4. 插入QualityReport
    5. 批量插入ValidationResult

    Args:
        document_id: 文档ID
        processing_result: 处理结果
        db: 数据库会话

    Returns:
        保存结果
    """
    try:
        # 1. 提取基本信息
        well_no = processing_result.get('well_info', {}).get('well_no')
        document_info = processing_result.get('document_info', {})
        extracted_fields = processing_result.get('extracted_fields', {})
        quality_metrics = processing_result.get('quality_metrics', {})
        validation_results = processing_result.get('validation_results', {})

        # 2. 创建/获取井记录
        well = WellCRUD.get_or_create_well(db, well_no)

        # 3. 创建文档记录
        document = DocumentCRUD.create_document(
            db=db,
            well_no=well_no,
            document_id=document_id,
            filename=document_info.get('filename'),
            file_path=document_info.get('file_path'),
            file_size=document_info.get('file_size'),
            category=document_info.get('category'),
            doc_category=document_info.get('doc_category')
        )

        # 4. 批量插入提取数据
        if settings.SAVE_EXTRACTED_DATA:
            ExtractedDataCRUD.batch_insert_extracted_data(
                db=db,
                document_id=document.id,
                well_no=well_no,
                extracted_fields=extracted_fields
            )

        # 5. 插入质量报告
        if settings.SAVE_QUALITY_REPORTS:
            QualityReportCRUD.create_quality_report(
                db=db,
                document_id=document.id,
                well_no=well_no,
                quality_metrics=quality_metrics,
                issues=quality_metrics.get('issues', []),
                suggestions=quality_metrics.get('suggestions', [])
            )

        # 6. 批量插入验证结果
        ValidationResultCRUD.batch_insert_validation_results(
            db=db,
            document_id=document.id,
            validation_results=extracted_fields  # 字段级别的验证
        )

        # 7. 更新文档状态
        DocumentCRUD.update_document_status(db, document_id, 'success')

        return {
            'success': True,
            'message': '数据已保存到数据库',
            'document_db_id': document.id,
            'well_db_id': well.id
        }

    except Exception as e:
        # 处理失败,更新文档状态
        DocumentCRUD.update_document_status(db, document_id, 'failed')
        db.rollback()
        raise e
```

```python
def _check_and_create_indexes(inspector):
    """检查并创建缺失的索引"""
    # 定义需要的索引
    required_indexes = {
        'wells': ['well_no', 'oilfield', 'block', 'status'],
        'documents': ['well_no', 'category', 'doc_category', 'status'],
        'extracted_data': ['document_id', 'well_no', 'field_name', 'confidence'],

    }

    for table_name, index_columns in required_indexes.items():
        if table_name in inspector.get_table_names():
            existing_indexes = [
                idx.get('column_names', [])
                for idx in inspector.get_indexes(table_name)
            ]

            for col in index_columns:
                # 检查索引是否存在
                index_exists = any(
                    col in existing_idx
                    for existing_idx in existing_indexes
                )

                if not index_exists:
                    # 创建索引
                    with engine.connect() as conn:
                        conn.execute(
                            text(f"CREATE INDEX idx_{col} ON {table_name} ({col})")
                        )
                        conn.commit()
```

## 七、总结

### 7.1 双向校验机制

```
┌─────────────────────────────────────────────────────────────┐
│                    双向校验架构                              │
└─────────────────────────────────────────────────────────────┘

1. 字段库 (config/field_schemas.py)
   ↓ 定义字段规范
   - 字段类型
   - 必填性
   - 数值范围
   - 单位

2. API Schema (schemas/models.py)
   ↓ 请求/响应校验
   - @validator('category')
   - @validator('target_fields')
   - @validator('extracted_fields')

3. 数据库模型 (db/models.py)
   ↓ 表结构定义
   - 表设计
   - 字段类型
   - 约束
   - 索引
   - 外键

4. 数据库表 (PostgreSQL)
   ↓ 数据持久化
   - 数据验证
   - 完整性检查
   - 事务保证

校验方向:
- → 字段库 → API Schema: 请求参数校验
- → API Schema → 字段库: 响应数据校验
- → 字段库 → 数据库模型: 字段映射验证
- → 数据库模型 → 字段库: 数据完整性验证
```

### 7.2 数据流转策略

**推荐: 混合模式**

1. **默认行为**: 保存到数据库
   - 优点: 数据持久化,可追溯,支持后续查询
   - 缺点: 响应稍慢
   - 配置: `save_to_db=true`

2. **可选行为**: 不保存到数据库
   - 优点: 响应快速
   - 缺点: 数据不持久
   - 配置: `save_to_db=false`

3. **批量处理**: 批量插入提高性能
   - 使用 `bulk_save_objects`
   - 事务保证一致性

### 7.3 关键要点

1. **一致性**: 字段库、API Schema、数据库模型保持一致
2. **校验**: 多层校验确保数据质量
3. **灵活性**: 提供配置选项控制数据持久化
4. **性能**: 批量插入、索引优化
5. **可追溯**: 完整的处理日志和审核记录
