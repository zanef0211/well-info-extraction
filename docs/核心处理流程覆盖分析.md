## 二、AI驱动的核心处理

### 需求描述

1. **文档类型识别与分类**: 采用LLM与关键词匹配算法,将文档自动归类至预定义文档分类类之中
2. **井号识别与去重**: 利用LLM和自然语言处理技术,从文档名称或内容中提取`井号(Well No)`。在多口井资料批量导入场景中,以井号作为文档聚合的唯一标识
3. **关键信息结构化提取**: 针对每一类文档,依据行业标准字段库,利用LLM模型,从非结构化文本中精准提取预定义的关键信息项

```python
class PPTParser:

        prs = self.Presentation(file_path)
```

```python
# 新建 models/wellno_extractor.py
class WellNoExtractor:
    """井号提取器和去重器"""

    def normalize_well_no(self, well_no: str) -> str:
        """归一化井号"""
        # 统一大写、去除空格、统一格式
        well_no = well_no.upper().strip()
        well_no = re.sub(r'\s+', '', well_no)
        return well_no

    def deduplicate_by_well_no(
        self,
        documents: List[Dict[str, Any]]
    ) -> Dict[str, List[Dict[str, Any]]]:
        """按井号去重并分组"""
        well_groups = {}

        for doc in documents:
            well_no = doc.get("WellNo", "")
            if not well_no:
                continue

            # 归一化
            normalized = self.normalize_well_no(well_no)

            # 分组
            if normalized not in well_groups:
                well_groups[normalized] = []
            well_groups[normalized].append(doc)

        return well_groups

    def get_unique_wells(
        self,
        documents: List[Dict[str, Any]]
    ) -> List[str]:
        """获取唯一井号列表"""
        normalized_set = set()
        for doc in documents:
            well_no = doc.get("WellNo", "")
            if well_no:
                normalized_set.add(self.normalize_well_no(well_no))
        return sorted(list(normalized_set))
```

### 3. 关键词匹配算法 (低优先级)

**当前状态**: ⚠️ 使用纯 LLM 分类,没有关键词匹配

**需求原因**: 关键词匹配可以作为 LLM 的补充,提高分类速度和准确性。

**实现建议**:

```python
class DocumentClassifier:
    def classify(
        self,
        document_text: str,
        metadata: Optional[Dict[str, Any]] = None,
        use_keywords_first: bool = True,
    ) -> ClassificationResult:
        """文档分类 - 关键词 + LLM 混合策略"""

        # ✅ 1. 关键词快速分类
        if use_keywords_first:
            keyword_result = self._classify_by_keywords(document_text, metadata)
            if keyword_result and keyword_result.confidence > 0.8:
                # 置信度高,直接返回
                return keyword_result

        # ✅ 2. LLM 精确分类
        llm_result = self._classify_by_llm(document_text, metadata)

        return llm_result

    def _classify_by_keywords(
        self,
        text: str,
        metadata: Dict
    ) -> Optional[ClassificationResult]:
        """基于关键词的分类"""

        # 定义关键词
        keyword_map = {
            "drilling": ["钻井", "钻进", "钻头", "井深", "钻压"],
            "logging": ["测井", "电测", "井径", "电阻", "声波"],
            "production": ["产量", "产油", "产气", "生产", "开采"],
            # ...
        }

        # 匹配关键词
        matched_types = []
        for doc_type, keywords in keyword_map.items():
            matches = sum(1 for kw in keywords if kw in text)
            if matches > 0:
                matched_types.append((doc_type, matches))

        if matched_types:
            # 选择匹配最多的
            best_type, matches = max(matched_types, key=lambda x: x[1])
            confidence = min(0.9, 0.6 + matches * 0.05)

            return ClassificationResult(
                document_type=best_type,
                confidence=confidence,
                reason=f"关键词匹配: {matches} 个关键词"
            )

        return None
```
