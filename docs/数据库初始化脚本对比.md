## ⚠️ 注意事项

### 1. 管理员密码

两个脚本使用的密码:
- PostgreSQL 管理员: `postgres` / `1437@2026`
- 应用数据库用户: `wellinfo` / `wellinfo2026`

**生产环境建议**: 修改默认密码!

### 2. 权限差异

**init_database_simple.py**: 仅授予连接权限
**init_database.py**: 授予完整权限(模式、序列、表)

如果使用简化脚本遇到权限问题,可以手动补充:

```sql
-- 连接数据库
psql -U postgres -d postgres

-- 授予完整权限
GRANT ALL ON SCHEMA public TO wellinfo;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON TABLES TO wellinfo;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON SEQUENCES TO wellinfo;
```

### 3. 初始用户

**init_database.py** 会创建管理员用户:
- 用户名: `admin`
- 密码: `admin123` (生产环境必须修改!)

**init_database_simple.py**: 不创建任何用户
