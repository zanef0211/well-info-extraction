#### 技术选型确认
- **OCR引擎**: PaddleOCR v4 (中文支持优秀,专业版对手写字体友好)
- **文档分类**: BGE-M3 Embedding (零样本) + BERT-wwm-ext Fine-tune (有标注数据)
- **井号识别**: 正则规则优先 + LLM兜底
- **字段抽取**: Qwen2.5-72B/DeepSeek-V3 (API) 或 Qwen2.5-14B-Instruct (私有化部署)
- **后端框架**: FastAPI
- **前端框架**: Vue 3 + TypeScript + Element Plus
- **数据库**: PostgreSQL 15
- **缓存**: Redis 7
- **向量库**: Milvus / Chroma
- **部署方式**: 支持API调用和私有化部署两种模式
